# resume-structure-guide - 简历结构指导技能

## 用途

帮助用户理解**通用简历结构**，适用于：
- 不同经验水平（应届/初级/中级/高级）
- 不同岗位（程序/策划/美术/测试/运营）
- 游戏行业及相关技术领域

## 触发词

- "简历结构"、"简历模板"、"简历怎么写"
- "简历应该包含"、"简历格式"、"简历框架"
- "优化简历结构"、"简历怎么写更好"

## 与 resume-optimizer 的区别

| 技能 | 用途 | 输出 |
|------|------|------|
| `resume-structure-guide` | 指导简历结构和写作原则 | 结构说明 + 模板 + 示例 |
| `resume-optimizer` | 优化具体简历内容并生成精美 HTML | 可直接使用的 HTML 简历 |

**使用建议**：
- 想了解"简历应该怎么写" → 用本技能
- 已有简历需要优化并生成精美版本 → 用 `resume-optimizer`

## 输出内容

1. 简历核心原则（回答 3 个问题）
2. 标准 6 部分结构详解
3. 不同经验水平的侧重点对比
4. 常见错误与修正
5. 万能模板（可直接套用）
6. 自检清单

## 示例用法

```
用户：简历应该包含哪些部分？
用户：简历结构怎么安排比较好？
用户：我是应届生，简历应该重点写什么？
用户：帮我看看这个简历结构有没有问题（附简历）
```
