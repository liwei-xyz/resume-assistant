# resume-assistant - 简历助手

> 🦐 专业、可编辑的 HTML 简历模板生成工具

## 📖 简介

**resume-assistant** 是一个帮助求职者快速创建专业简历的工具，特别针对游戏行业服务器开发岗位优化。

**核心理念**：
- ✅ 一页纸原则 - HR 筛选简历平均只要 6-10 秒
- ✅ 结构清晰 - 项目简介/技术架构/主要工作 三分法
- ✅ 量化成果 - 用数据说话，不用形容词
- ✅ 可编辑 HTML - 浏览器直接点击修改，无需安装软件

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| 📝 可编辑 HTML | 所有文字可点击编辑，所见即所得 |
| 📷 照片上传 | 右上角预留照片区域（可选） |
| 📄 导出 PDF | 一键导出专业 PDF 简历 |
| 🏷️ 自动命名 | 按 `岗位_名字.html` 格式命名 |
| 📱 一页布局 | 紧凑排版，符合 HR 筛选习惯 |

## 🚀 快速开始

### 方式 1：作为 OpenClaw 技能使用

```bash
# 克隆到 OpenClaw 技能目录
git clone https://github.com/你的用户名/resume-assistant.git ~/.openclaw/workspace/skills/resume-assistant

# 在 OpenClaw 中触发
对我说："帮我写简历"
```

### 方式 2：直接使用 HTML 模板

1. 下载 `resume-template.html`
2. 用浏览器打开
3. 点击文字编辑内容
4. 点击"导出 PDF"保存

## 📋 简历结构

```
基本信息
├── 姓名
├── 联系方式（手机/邮箱/城市/GitHub）
└── 照片（可选）

个人概述（4 点）
├── 工作经验
├── 项目经历
├── 技术能力
└── 团队协作

工作经历（倒序）
├── 公司 1 | 职位     时间
│   └── 3-4 条成果描述
└── 公司 2 | 职位     时间
    └── 3-4 条成果描述

项目经历（倒序）
├── 项目 1（品类）| 角色     时间 | 团队规模
│   ├── 项目简介（1-2 点）
│   ├── 技术架构（后端 + 数据库）
│   └── 主要工作（4-6 条）
└── 项目 2...

专业技能
├── 编程语言
├── 网络编程
├── 并发模型
├── 架构能力
├── 数据库
└── 工具平台

教育背景
└── 学校 | 专业 | 学历     时间
```

## 📝 写作原则

### ✅ 正确写法
```
- 负责 MMORPG 手游核心战斗服务器架构设计，支持 3000+ 并发
- 设计基于 UDP(KCP) 的高性能网络传输层，响应时间缩短 30%
- 单服承载从 1000 人提升至 3000 人
```

### ❌ 错误写法
```
- 负责服务器开发（无成果）
- 优化了系统性能（无数据）
- 熟悉多种技术（太模糊）
```

## 🎯 适用人群

- **应届/初级**：突出学习能力、项目热情、GitHub 作品
- **中级 (3-5 年)**：突出技术深度、独立负责能力、量化成果
- **高级 (5 年+)**：突出架构能力、业务价值、团队贡献
