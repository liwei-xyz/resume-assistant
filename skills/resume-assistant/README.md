# resume-assistant - 简历助手技能

## 用途

生成专业、可编辑的 HTML 简历模板，帮助用户快速创建一页纸简历。

## 触发词

- "帮我写简历"
- "简历模板"
- "简历结构"
- "优化简历"
- "生成简历"
- "简历怎么写"

## 功能特点

1. **可编辑 HTML**：所有文字可点击编辑
2. **照片上传**：右上角预留照片区域（可选）
3. **导出 PDF**：一键导出专业 PDF 简历
4. **自动命名**：按 `岗位_名字.html` 格式命名
5. **一页纸**：紧凑布局，符合 HR 筛选习惯

## 文件结构

```
resume-assistant/
├── SKILL.md            # 技能说明
└── resume-template.html  # HTML 模板
```

## 使用方法

1. 用户触发技能
2. 复制模板到工作区
3. 浏览器打开供用户编辑
4. 指导用户填写和优化
5. 导出 PDF 投递简历

## 输出示例

```
简历模板已生成并在浏览器打开！

使用方法：
1. 点击任意文字即可编辑
2. 右上角照片区域可点击上传（可选）
3. 编辑完成后点击"导出 PDF"或"保存修改"
```

## 简历结构

1. 基本信息（姓名 + 联系方式 + 照片）
2. 个人概述（4 点：经验、项目、技术、协作）
3. 工作经历（倒序，量化成果）
4. 项目经历（分 3 部分：简介、架构、工作）
5. 专业技能（分类清晰）
6. 教育背景（简洁）

## 写作原则

- ✅ 动词开头：负责、主导、搭建、优化
- ✅ 量化成果：QPS、DAU、百分比
- ✅ 业务价值：支撑多少用户、创造多少收入
- ❌ 不编造：内容必须真实
- ❌ 不模糊：用数据代替形容词
